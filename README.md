# Mcsam's Blog
Blogging API written with the FastAPI framework in python3.

## Features of API
1. Registering an account.
2. Logging into the account.
3. Deleting an account.
4. Create blog posts.
5. Updating blog posts.
6. Deleting blog posts.
7. Commenting on blog posts.
8. Updating  comments.
9. Deleting comments.
